Todo List
===================


Project Description
-------------------

This app goes beyond the basic features of a typical todo list, offering advanced functionality such as subtask creation, priority levels, filtering, and a completed items list. See the Features section below for the full list.


Run this app live
-------------------


Run Locally
-------------------

1. Clone the repository  
2. Install dependencies using `npm install`  
3. Start the development server with `npm start`

Features
-------------------
- Built with React and Redux for state management and user interaction. 
- Create todos by entering text, time, and selecting a priority level.
- View and manage the list of created todos
    - Display text and time for each item
    - Priority-based color coding (green, orange, red)
    - Drag-and-drop reordering
    - Completion percentage bar that updates as items are completed
- Create and manage subtasks for each todo
- View completed items separately


Tech Stack
-------------------
- HTML
- CSS
- JavaScript
- React
    - Redux
